# DukanKonnect - Compose Multiplatform Service Booking App 

**DukanKonnect** is a fully functional Compose Multiplatform (CMP) application designed to seamlessly connect local vendors and service providers with customers. 

This repository serves as a **showcase of CMP architecture** and a **starter reference** for developers looking to build cross-platform mobile apps (Android & iOS) with a shared Kotlin codebase and 100% shared UI.

## ✨ Key Features

* **Service Discovery & Booking:** Browse local service providers and book appointments.
* **Time & Location Selection:** Interactive UI for users to select precise service times and locations.
* **Integrated Payments:** End-to-end payment flow simulation.
* **Booking Management:** Dedicated dashboard for users to track, edit, or cancel active bookings.
* **Cross-Platform Notifications:** Real-time push notification handling implemented for both Android and iOS.
